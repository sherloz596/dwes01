Instalación

Para instalar hugo en su última versión mejor instalar con el comando snap.

snap install hugo

Si necesitamos soporte Sass/SCSS:

snap install hugo --channel=extended

Para crear un nuevo sitio:

hugo new site <nombre_del_sitio>
